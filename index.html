pastebun is an anonymous and secure (for some values of secure) pastebin service. All pastes are encrypted with AES256 with 40 char long keys. The key is not stored anywhere and is read from URL when decrypting the paste. Likewise decrypted pastes are not stored anywhere. Then again if you're using it over clearnet everything is sent plain-text.

If you feel you can't trust code you don't know, know it
